LIVE SAVE — v2 (FIXED: unlimited saves, white logo backgrounds, popup forms)

=== WHAT WAS FIXED ===

1. SAVE FAILED / LIMITED SAVES — ROOT CAUSE FIXED
   The old version sent the entire state (including all uploaded images as base64)
   in one JSON POST body, which hit Netlify's 6MB function limit after a few images.
   
   The new Netlify Function (portfolio-state.js) now SPLITS binary blobs out of
   the state before saving:
     - Each logo, profile photo, and attachment is stored separately in Netlify Blobs
     - The JSON body sent to the function is tiny (text-only)
     - On load, blobs are re-assembled transparently
   Result: You can now save UNLIMITED times with any number of images.

2. WHITE LOGO BACKGROUNDS
   Logo squares now have a white (#ffffff) background with object-fit:contain,
   so transparent PNG logos display cleanly with no dark background showing through.

3. POPUP FORM MODALS
   Add Item and Edit Item now open a proper popup form modal (no more browser prompts)
   with labeled fields specific to each section:
     - Education: School, Year/Degree, Achievements
     - Work: Position, Duration/Org, Responsibilities
     - Trainings: Certificate Title, Date/Issuer, Description
     - Research: Title, Year/Type, Abstract
     - etc.
   Add Attachment also has a proper file-picker modal with title, overview, and file zone.
